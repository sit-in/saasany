# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in Saasany, please report it responsibly:

1. **Do NOT open a public issue**
2. Email: security@saasany.com (or use GitHub's private vulnerability reporting)
3. Include: description, steps to reproduce, potential impact

We will respond within 48 hours and provide a fix timeline.

## Scope

This policy covers the Saasany template code. Third-party services (Stripe, Neon, OpenAI, etc.) have their own security policies.
