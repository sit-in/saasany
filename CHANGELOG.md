# Changelog

## 0.1.0 (2026-03-21)

### Features

- **Authentication** - Better Auth with email/password and GitHub OAuth
- **Landing Page** - Hero section with dashboard preview, features grid, pricing cards
- **Dashboard** - Collapsible sidebar, stats overview, settings, billing management
- **Payments** - Stripe Checkout, webhooks, customer portal, subscription management
- **AI Chat** - Streaming chat with Vercel AI SDK v6
- **AI Generation** - Text generation with preset templates
- **API Keys** - Create, list, and revoke API keys
- **Blog** - MDX-based blog system with i18n support
- **Admin Panel** - User management, subscription stats, role-based access
- **Email** - Welcome and subscription confirmation templates via Resend
- **Internationalization** - English and Chinese translations
- **Dark Mode** - System-aware theme switching with next-themes
- **SEO** - Sitemap, robots.txt, OG image generation, metadata API
- **Legal** - Privacy Policy and Terms of Service pages
- **Error Handling** - Custom error, 404, and loading pages
